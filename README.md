## Current Working Shortlist

The strongest current idea clusters are:

1. Corporate Wellness / Strava for Work
2. Active Travel / Adventure / Active Experiences
3. Race & Event Marketplace

Ideas worth keeping alive as non-obvious alternatives:

- Creator Economy
- Local Experiences Marketplace
- Social sports / Padel-Tennis layer

## Working Principle

The team should not commit only because an idea looks "most business-like" or "most creative."

The decision should balance:

- strategic fit with Strava
- monetization clarity
- strength of market validation
- ease of prototyping
- wow-effect for the final jury
