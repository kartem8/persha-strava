## Блоки по всьому проєкту

| Блок | Що входить | Основний результат |
|---|---|---|
| `Lead / PM` | Координація команди, дедлайни, sync-и, контроль прогресу по всіх блоках, збірка фінальної логіки | Master outline, список рішень, контроль дедлайнів |
| `Final presentation / storytelling` | Структура deck, наратив, логіка слайдів, формулювання тез, розподіл хто що презентує, Q&A prep | Фінальна структура презентації, storytelling, speaking flow |
| `Strava product analysis` | Аналіз поточного продукту Strava, core strengths, user behavior, monetization | Strava summary, why Strava can win |
| `Competitors + market research` | Прямі конкуренти, adjacent markets, category scan, trends | Competitor scan, market map |
| `Hypotheses + scoring + selection` | Longlist, критерії, scoring matrix, top-3, фінальний вибір | Longlist, shortlist, chosen hypothesis |
| `Validation + evidence + mentor prep` | Джерела, перевірка цифр, premium tools, питання до ментора, plan validation | Evidence table, mentor questions, validation plan |
| `Interviews + user insights` | Пошук респондентів, сценарій інтерв’ю, проведення 5-10 розмов, збір повторюваних болей та інсайтів | Interview notes, insight summary, quotes / patterns |
| `Product adaptation + prototype` | Product concept, feature adaptation, user flows, prototype, demo logic | Product concept, prototype, demo flow |
| `GTM + monetization + unit economics` | GTM strategy, funnel, monetization, unit economics, basic finance | GTM block, revenue logic, financial model |
