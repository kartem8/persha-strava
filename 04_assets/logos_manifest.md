# Competitor Logos Manifest

## Files

- `garmin.svg` -> Garmin Connect
- `nike.svg` -> Nike Run Club
- `adidas.svg` -> adidas Running
- `runkeeper.svg` -> ASICS Runkeeper
- `underarmour.svg` -> Under Armour / MapMyRun fallback
- `mapmyrun.png` -> MapMyRun favicon fallback
- `komoot.svg` -> komoot
- `coros.png` -> COROS favicon fallback

## Notes

- SVGs from `simpleicons` are clean and best for presentation use.
- `COROS` and `MapMyRun` currently use favicon fallbacks because a clean simple SVG source was not available in the quick asset pass.
- If Lovable cannot use images reliably, these can still be used later in the exported project or replaced with branded text chips.
